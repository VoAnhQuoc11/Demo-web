# Ảnh chụp sản phẩm cần nộp

Chạy `npm run dev`, mở trình duyệt và chụp các màn hình sau, lưu vào `docs/screenshots/`:

1) Home: trang chủ sau khi load.
2) Auth - Login: form đăng nhập, kịch bản sai mật khẩu và đúng.
3) Auth - Register: form đăng ký thành công.
4) Blog List: liệt kê, tìm kiếm (search), sắp xếp (sort), phân trang.
5) Blog Detail: xem chi tiết 1 bài.
6) Blog Create: tạo bài, upload ảnh.
7) Blog Edit: cập nhật bài.
8) Blog Delete: xác nhận xoá (trước/sau).

Đặt tên file gợi nhớ, ví dụ:
- `01-home.png`
- `02-login.png`
- `03-register.png`
- `04-blog-list-search-sort.png`
- `05-blog-detail.png`
- `06-blog-create-upload.png`
- `07-blog-edit.png`
- `08-blog-delete.png`

> Nếu cần, kèm thêm ảnh flow auth/refresh token hoặc sơ đồ luồng blog CRUD.
